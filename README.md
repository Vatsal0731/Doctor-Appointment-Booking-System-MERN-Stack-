# Doctor-Appointment-Booking-System-MERN-Stack-
## How to Set Up & Run the Project

> **Order of execution:**\
> Backend → Frontend → Admin

## ✅ Prerequisites

-   Install Node.js: https://nodejs.org/en/download/

## ✅ Backend Setup

1.  Open the project folder in VS Code.

2.  Right-click the **backend** folder → **Open in Integrated Terminal**

3.  Install dependencies:

        npm install

4.  Run backend:

        npm run server

## ✅ Frontend Setup

1.  Right-click **frontend** folder → **Open in Integrated Terminal**

2.     npm install

3.     npm run dev


## ✅ Admin Panel Setup

1.  Right-click **admin** folder → **Open in Integrated Terminal**

2.     npm install

3.     npm run dev
